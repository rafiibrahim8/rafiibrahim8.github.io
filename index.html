Projects

iSteg

A little steganography. Hiding text or file inside an image.

iSteg is a program written in Java that can hide any file inside an image. It modifies the least significant bits (LSB) of an image to store data. iSteg has both a GUI and a command-line version, and also supports encryption. With iSteg, your private files are now safer than ever.

Stack: Java, Steganography, AES256, Swing

Repository

asteg

Another steganography project. But this time with ear.

Asteg is a python script that can be used to hide files inside an audio file without compromising any quality. It uses high-frequency components of audio to do so. Asteg uses On-Off Keying (OOK) to modulate the wave according to the information of the file.

Stack: Python, Steganography, OOK
